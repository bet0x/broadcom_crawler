# Remove a License from a vCenter Instance

To remove a license from a vCenter instance, you must assign a new license to the instance.

For more information about assigning a license to a vCenter instance, see [Assign a Primary License to a vCenter Instance or a VCF Instance.](/us/en/vmware-cis/vcf/vcf-9-0-and-later/9-0/licensing/assign-a-licenses-to-a-vcenter-instance.html)
