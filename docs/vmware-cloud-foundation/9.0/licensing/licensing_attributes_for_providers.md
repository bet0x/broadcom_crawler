# Licensing Attributes for Providers

On the Licensing tab of the VCF Business Services console, you can see the attributes of the licenses you manage.

## Licensing Attributes

| Component | Description |
| --- | --- |
| Commit Capacity | The committed capacity in the signed commit contract. Can be in number of cores or TiBs. |
| Total Licenses | Total number of the created 9.0 licenses. |
| Consumed Capacity | The consumed capacity contains data for both non-running and running workloads. If VCF usage meter manages the environment on a running workload and the consumed capacity exceeds the commit capacity, you might get billed for over use. |
