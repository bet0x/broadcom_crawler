# View Product License Usage in the VCF Business Services console

In the VCF Business Services console, you can view aggregated data for your complete license usage for all your products and licenses in the unit of measure for the specific license. You can use the analytics data to monitor license usage and plan your license management.

You can also export and view the usage analytics data for product aggregate usage, version 9+ license usage, and pre-version 9 license usage. The export file is available in CSV and JSON formats.

1. Log in to the VCF Business Services console (vcf.broadcom.com).
2. Navigate to License ManagementUsage Analytics.
3. In the Usage by Product pane, click the product for which you want to view detailed license usage.

   The Usage Details page for the selected product appears.
4. If you want to view usage analytics for a specific license, use the Select License drop-down menu.
5. In the Usage History pane, you can view maximum license usage for a specific aggregation period.

   Use the filters to view maximum license usage in monthly or daily periods.
