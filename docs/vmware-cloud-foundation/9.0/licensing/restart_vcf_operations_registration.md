# Restart VCF Operations Registration

If you register the VCF Operations in the wrong site or tenant, you can restart the VCF Operations registration.

Before you restart the VCF Operations registration, you must remove any licenses added to the VCF Operations instance, delete the VCF Operations instance, and then re-register in the correct tenant.

1. In VCF Operations, navigate to License ManagementRegistration.
2. In the upper right corner, click Actions.
3. Click Restart Registration.
