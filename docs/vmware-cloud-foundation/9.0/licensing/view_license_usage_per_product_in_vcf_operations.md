# View License Usage Per Product in VCF Operations

In your VCF Operations instance, you can view aggregated data for your license usage for products and for specific licenses, based on the licenses added to this VCF Operations instance. You can use the analytics data to monitor license usage and plan your license management.

1. In the VCF Operations instance, navigate to License ManagementUsage Analytics.
2. In the Usage by Product pane, click the product for which you want to view detailed license usage. The Usage Details page for the selected product appears.
3. If you want to view usage analytics for a specific license, use the Select License drop-down menu.
4. If you want to view usage analytics for a specific vCenter instance, use the vCenter drop-down menu.

   The list of vCenter instances includes only the instances to which you have assigned licenses from this VCF Operations instance. If you select a specific license to view data for, the list of vCenter instances contains only the vCenter instances to which this license is assigned.
5. In the Usage History pane, you can view maximum license usage for a specific aggregation period.

   Use the filters to view maximum license usage in monthly or daily periods.
