## Introduction

Async releases relate to Express Patches of components released without a corresponding VCF release or third-party deliverables that are released async by other companies in the VCF ecosystem, such as Kubernetes software distributions, or the Async Patch Tool, which allows you to apply critical patches to certain VCF components outside of VCF releases.
