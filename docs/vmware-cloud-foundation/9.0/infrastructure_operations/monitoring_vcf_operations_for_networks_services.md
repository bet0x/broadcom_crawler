# Monitoring VCF Operations for networks Services

You can monitor the VCF Operations for networks services with the VMware Infrastructure Health.

## Resources Monitored from VCF Operations for networks

VMware Infrastructure Health monitors the health of the following VCF Operations for networks services:

- CollectorMain
- FoundationDB
- IpfixProcessor
- Nfcapd
- NetopaCollector
- ntpsec
- Nginx
- SwitchTelemetryS
