# Monitoring SDDC Manager Services

You
can monitor the health of SDDC Manager services with VMware Infrastructure Health.

## Resources Monitored for SDDC Manager

VMware Infrastructure Health starts collecting the health of the
following SDDC Manager services:

- COMMON\_SERVICES
- DOMAIN\_MANAGER
- LCM
- OPERATIONS\_MANAGER
- SDDC\_MANAGER\_UI
