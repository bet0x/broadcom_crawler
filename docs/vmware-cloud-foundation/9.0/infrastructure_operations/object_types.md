# Object Types

VCF Automation brings in cloud accounts and their relationships from VCF Automation for VM Apps Organization tenant into VCF Operations for operational analysis. You can use the following items in the virtual infrastructure as object types in VCF Operations.

- Cloud Zone
- Blueprint
- Project
- Deployment
- Cloud Account
- User
- Organization
- Cloud Automation Services World
