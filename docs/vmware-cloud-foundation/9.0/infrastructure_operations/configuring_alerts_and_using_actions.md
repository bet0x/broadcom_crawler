# Configuring Alerts and Using Actions

Alerts are a way to be watchful of anything that is new or an issue that could be potentially dangerous to your environment. Whenever there is a problem in the environment, the alerts are generated. You can also create new alert definitions so that the generated alerts inform you about the problems in the monitored environment. In VCF Operations , alerts and actions play key roles in monitoring the objects.
