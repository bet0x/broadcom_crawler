# vSphere Supervisor Monitoring

The management pack introduced in VMware Cloud Foundation 9.0 helps you monitor the resource utilization and performance of vSphere Supervisor and VKS Clusters and resources using VCF Operations. This is an internal, native management pack which you enable when configuring a vCenter integration.
