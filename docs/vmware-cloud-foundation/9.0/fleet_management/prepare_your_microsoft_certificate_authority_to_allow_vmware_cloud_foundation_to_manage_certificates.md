# Prepare Your Microsoft Certificate Authority to Allow VMware Cloud Foundation to Manage Certificates

If you plan to configure a Microsoft Certificate Authority for use with VMware Cloud Foundation, there are some steps you must perform before you can do so.

In order to generate signed certificates the Microsoft Certificate Authority must be configured to allow integration with VMware Cloud Foundation.
