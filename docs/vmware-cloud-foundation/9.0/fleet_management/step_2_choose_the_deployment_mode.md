# Step 2: Choose the Deployment Mode

After you have selected a VCF Instance, you must choose a deployment mode for the VCF Identity Broker configuration. The options available are embedded or appliance. For more information about the VCF Single Sign-On deployment modes, see [Deployment Modes of the VCF Identity Broker](/us/en/vmware-cis/vcf/vcf-9-0-and-later/9-0/fleet-management/what-is/deployment-models-for-sso.html).

**Prerequisite**

- Complete [Step 1: Select a VCF Instance](/us/en/vmware-cis/vcf/vcf-9-0-and-later/9-0/fleet-management/what-is/setting-up-sso/select-a-vcf-instance.html).

**Procedure**

1. From the Enable Single Sign-On page, click the Start button against the Choose deployment mode option.
2. Select either Identity broker (embedded) or Identity broker appliance. For information about deploying a VCF Identity Broker (appliance), see [Deploying VCF Identity Broker](/us/en/vmware-cis/vcf/vcf-9-0-and-later/9-0/deployment/deploying-a-new-vmware-cloud-foundation-or-vmware-vsphere-foundation-private-cloud-/manual-deployment-of-components-to-complete-your-vcf-platform/installing-vcf-identity-broker.html#GUID-0eafc123-11e5-4983-84bf-95d5d24c60ad-en_id-915ca638-7020-4655-d9aa-6cdc9909d6c3).
3. Click Next.
