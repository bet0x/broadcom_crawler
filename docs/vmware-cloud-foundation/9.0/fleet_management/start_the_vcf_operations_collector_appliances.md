# Start the VCF Operations Collector Appliances

Start the VCF Operations collector nodes in the management domain by using the vSphere Client.

1. Log in to vCenter for the management domain at https://<vcenter\_fqdn>/ui as a user with the Administrator role.
2. In the VMs and Templates inventory, locate a VCF Operations collector appliance.
3. Right-click theVCF Operations collector appliance and select PowerPower On.
4. In the configuration dialog box, click Yes.
5. Repeat the steps on the remaining VCF Operations collector appliances.
