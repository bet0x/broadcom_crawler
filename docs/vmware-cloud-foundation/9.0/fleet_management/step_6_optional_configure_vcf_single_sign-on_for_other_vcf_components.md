# Step 6: (Optional) Configure VCF Single Sign-On for other VCF Components

Configure VCF Single Sign-On for other VCF Components such as VCF Operations HCX, VCF Operations for networks, VCF Operations orchestrator, and VCF Operations for logs.

- To configure VCF Single Sign-On for VCF Operations HCX, [Configure VCF Single Sign-On.](/us/en/vmware-cis/vcf/vcf-9-0-and-later/9-0/workload-mobility/vmware-hcx-user-guide-vcf-9-0/managing-system-settings/configure-vcf-sso.html#GUID-bca5a24e-7e91-424a-acf0-a64a05977bd6-en_id-35b7803b-d756-4784-a8ab-8b20fc9541d6)
- To configure VCF Single Sign-On for VCF Operations for networks, see [Configuring VCF SSO for VCF Operations for Networks](/us/en/vmware-cis/vcf/vcf-9-0-and-later/9-0/infrastructure-operations/network-operationss/configuration/configuring-vcf-sso-for-operations-for-networks.html).
- To configure VCF Single Sign-On for VCF Operations for logs, see [Configuring Single Sign-On for VCF Operations for logs](/us/en/vmware-cis/vcf/vcf-9-0-and-later/9-0/infrastructure-operations/log-analysis/getting-started-with-log-analysis/configuring-vcf-logs-sso.html#GUID-3d35589d-119e-4ed4-81c4-cf634bbff5dd-en_id-f8bdb754-792e-444c-a7f6-31031f995cd6).
- To configure Single Sign-On for CLI tools like PowerCLI, see [Using VCF Single Sign-On to access vCenter using PowerCLI](/us/en/vmware-cis/vcf/vcf-9-0-and-later/9-0/fleet-management/what-is/setting-up-sso/configure-vmware-cloud-foundation-sso--as-a-client-for-other-components/vcf-single-sign-on-for-powercli.html#GUID-8d03903c-dc78-472b-86ab-1abc6faaa2cb-en_id-9e711147-5c4d-4947-87e3-0ddcf3984a77).
