# Edit a Tag in VCF Operations

You can edit the description of any tag in VCF Operations. You cannot change the tag names.

- Verify that you have TagsManagepermissions in VCF Operations.

1. Log in to VCF Operations.
2. From the left navigation menu, click Fleet Management, and click Tags.
3. In the table with available categories, click the double arrow button next to the category that contains the tag you want to edit.

   A table with all tags added to this category appears on the right.
4. In the table with available tags, select the check box next to the tag you want to edit.

   You can edit only one tag at a time.
5. Click Edit.
6. Enter a new description for the tag, and click Save.
