# Shut Down VCF Operations for Logs

Shut down the VCF Operations for logs appliances in the management domain by using the vSphere Client.

1. Log in to vCenter for the management domain at https://<vcenter\_fqdn>/ui as a user with the Administrator role.
2. In the VMs and Templates inventory, locate a VCF Operations for logs appliance.
3. Right-click the VCF Operations for logs appliance and select PowerShut down Guest OS.
4. In the confirmation dialog box, click Yes.
5. Repeat the steps for the remaining VCF Operations for logs appliances.
