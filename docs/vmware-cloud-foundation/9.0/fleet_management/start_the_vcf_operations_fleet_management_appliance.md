# Start the VCF Operations Fleet Management Appliance

Start the VCF Operations fleet management appliance in the management domain from the vSphere Client.

1. Log in to vCenter for the management domain at https://<vcenter\_fqdn>/ui as a user with the Administrator role.
2. In the VMs and Templates inventory, locate the VCF Operations fleet management appliance.
3. Right-click the VCF Operations fleet management appliance and select PowerPower On.
4. In the confirmation dialog box, click Yes.
