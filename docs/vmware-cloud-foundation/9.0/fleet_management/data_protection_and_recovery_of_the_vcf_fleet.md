# Data Protection and Recovery of the VCF Fleet

You can use VMware Live Site Recovery and VMware vSphere Replication to provide support for planned migration or disaster recovery of the VCF fleet and its managed VCF Instances.

For more information about data protection and recovery of the VCF fleet, see the [Site Protection and Recovery for VMware Cloud Foundation](https://techdocs.broadcom.com/bin/gethidpage?ux-context-string=site_protection&appid=vvs-9-X&language=&format=rendered) validated solution.
