# Shut Down the VCF Operations Collector Appliances

Shut down the VCF Operations collector nodes in the management domain of the VCF Instance by using the vSphere Client.

1. Log in to vCenter for the management domain at https://<vcenter\_fqdn>/ui as a user with the Administrator role.
2. In the VMs and Templates inventory, locate a VCF Operations collector appliance.
3. Right-click the appliance and select PowerShut down Guest OS.
4. In the confirmation dialog box, click Yes.
5. Repeat the steps for the remaining VCF Operations collector appliances.
