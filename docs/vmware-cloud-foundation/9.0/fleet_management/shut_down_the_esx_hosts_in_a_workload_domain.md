# Shut Down the ESX Hosts in a Workload Domain

You follow specific steps according to the principal storage of the clusters in the workload domain.

## Save the Credentials for the ESX Hosts and vCenter for the Workload Domain

Before you shut down the workload domain, get the credentials for the workload domain hosts and vCenter and save them. You might need these credentials to shut down the ESX hosts and then to start them and vCenter back up.
