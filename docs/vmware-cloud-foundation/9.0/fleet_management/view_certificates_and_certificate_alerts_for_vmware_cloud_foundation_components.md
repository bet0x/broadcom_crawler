# View Certificates and Certificate Alerts for VMware Cloud Foundation Components

You can view detailed information, including alerts, about the certificates for all VMware Cloud Foundation components from the VCF Operations console.

The certificate management UI includes filtering options that can help you find certificates based on type, status, component, and so on.

1. Log in to the VCF Operations console at https://<vcf\_operations\_fqdn> with a user assigned the Administrator role.
2. Click Fleet ManagementCertificates.
3. Click VCF Management to view certificate information for the VCF Management components.
4. Click VCF Instances and click a VCF Instance name to view certificate information for the components in the selected VCF Instance.

   Use the Show ESX Hosts toggle to show or hide ESX hosts.
5. Click VCF Instances and click a VCF domain name to view certificate information for the components in the selected VCF domain.

   Use the Show ESX Hosts toggle to show or hide ESX hosts.
6. Click Certificate Alerts to view certificate-related alerts.
