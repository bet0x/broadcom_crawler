# Move ESX Host into Selected Fault Domain in vSAN Cluster

You can move an ESX host into a selected fault domain in the vSAN cluster.

1. In the vSphere Client, navigate to the cluster.
2. Click the Configure tab.
3. Under vSAN, click Fault Domains.
4. Click and drag the ESX host that you want to add onto an existing fault domain.

   The Add Hosts To Domain dialog appears.
5. Click Move.

   The selected ESX host appears in the fault domain.
