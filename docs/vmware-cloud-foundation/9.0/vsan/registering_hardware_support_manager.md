# Registering Hardware Support Manager

You must register HSM with PHM that resides within the vSAN management service on the vCenter using the vendor management service.

For detailed information about registering hardware support managers, refer to the respective OEM-provided documentation and see [Deploying Hardware Support Managers](https://techdocs.broadcom.com/us/en/vmware-cis/vsphere/vsphere/9-0/managing-host-and-cluster-lifecycle/using-images-to-install-and-update-esxi-hosts-and-clusters/firmware-updates.html#GUID-B001135C-59AF-4CFD-9B73-94648B41D1A0-en) in the Managing Host and Cluster Lifecycle guide.
