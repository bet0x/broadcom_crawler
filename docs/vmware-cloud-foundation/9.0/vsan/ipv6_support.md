# IPv6 Support

vSAN supports IPv6.

vSAN supports the following IP versions.

- IPv4
- IPv6
- Mixed IPv4/IPv6. Use mixed mode when migrating your vSAN cluster from IPV4 and IPv6

For more information about using IPv6, consult with your network vendor.
