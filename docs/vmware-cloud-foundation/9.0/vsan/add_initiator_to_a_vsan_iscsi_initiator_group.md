# Add Initiator to a vSAN iSCSI Initiator Group

You can add a member to the vSAN iSCSI initiator group.

1. In the vSphere Client, navigate to the cluster.
2. Click the Configure tab.
   1. Under vSAN, click iSCSI Targets.
   2. Click the Initiator Groups tab.
   3. In the Accessible Targets section, click Add. The Add Initiators dialog box is displayed.
   4. Enter an initiator name in the Member initiator name field.
   5. Click Add to add a member to the initiator group.
3. Click Add.

   The vSAN iSCSI initiator is created and listed.
