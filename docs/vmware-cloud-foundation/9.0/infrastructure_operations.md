# Infrastructure Operations

Use **VCF Operations** as a VI administrator to optimize and monitor compute, storage, and network of your VCF platform. These operational capabilities include performance optimization, visibility and observability of infrastructure and operations, global configuration management, monitoring, troubleshooting, log analytics, end-to-end network monitoring, energy efficiency monitoring, and other.
