# Multi Tenancy Consumption Patterns

This section describes the multi-tenancy consumption patterns aligned with the tenancy deployment models described in the 'Tenancy Deployment Models" section. It explains the different ways multiple organizations or tenants can consume shared compute, memory, storage, and networking resources, depending on the chosen tenancy deployment model.
