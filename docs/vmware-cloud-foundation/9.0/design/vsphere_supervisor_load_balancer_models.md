# vSphere Supervisor Load Balancer Models

vSphere Supervisor requires selecting an appropriate load balancer for the backing workload networking type. The choice of load balancer includes NSX Load Balancer, Avi Load Balancer and Foundation Load Balancer. This section describes the different load balancer models.

- [Supervisor NSX Load Balancer Model](/us/en/vmware-cis/vcf/vcf-9-0-and-later/9-0/design/design-library/self-service-iaas-deployment-models/vsphere-supervisor-load-balancer-models/supervisor-nsx-load-balancer-model.html)
- [Supervisor Avi Load Balancer Model](/us/en/vmware-cis/vcf/vcf-9-0-and-later/9-0/design/design-library/self-service-iaas-deployment-models/vsphere-supervisor-load-balancer-models/supervisor-avi-load-balancer-model.html)
- [Supervisor Foundation Load Balancer Model](/us/en/vmware-cis/vcf/vcf-9-0-and-later/9-0/design/design-library/self-service-iaas-deployment-models/vsphere-supervisor-load-balancer-models/supervisor-foundational-load-balancer-model.html)
