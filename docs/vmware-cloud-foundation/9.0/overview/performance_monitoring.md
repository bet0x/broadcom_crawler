# Performance Monitoring

Performance monitoring ensures applications have continuous access to resources and AI-driven troubleshooting and remediation enables faster time to resolution of issues while keeping costs down. Manage, monitor and troubleshoot your applications by collecting application metrics from your virtual machines.

## Management Packs

A VCF Operations management pack, is a software package that allows you to monitor, manage, and troubleshoot infrastructure by collecting metrics, events, and alerts from the integrated system.

- Marketplace Plug-in

  Starting with VCF Operations 9.0, you can download and install integrations directly from within VCF Operations. For more information, see the [Installing Integrations in a Connected Environment](/us/en/vmware-cis/vcf/vcf-9-0-and-later/9-0/infrastructure-operations/extending-monitoring-capabilities/installing-and-configuring-management-packs-in-vcf-operations-from-vmware-marketplace/copy-of-about-vrealize-operations-management-packs/installing-and-configuring-integrations-in-vcf-operations/copy-of-installing-management-packs-in-vrealize-operations.html) documentation.
- Custom Management Pack

  You can create a custom management pack in VCF Operations. You can perform the following tasks:

  - Design, test, and install custom management packs.
  - Support for JSON and XML REST APIs.
  - Extend existing objects with new metrics and relationships.

    For more information, see the [Management Pack Builder](/us/en/vmware-cis/vcf/vcf-9-0-and-later/9-0/infrastructure-operations/extending-monitoring-capabilities/cloud-foundation-operations-configuration-guide-management-pack-builder.html) documentation.
- Log information is available for the vSphere, vSAN, VCF Operations, and VCF Automation management packs.
