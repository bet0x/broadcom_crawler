# Download VCF Core Component Binaries From an Online or Offline Depot

After you connect SDDC Manager to an online or offline depot, you can download upgrade, patch, and install binaries and use those binaries to update or deploy VCF domains.

Connect to an online or offline depot.

- [Connect to an Online Depot for a VCF Instance](/us/en/vmware-cis/vcf/vcf-9-0-and-later/9-0/lifecycle-management/lifecycle-management-of-vcf-core-components/configure-an-online-depot-for-a-vcf-instance.html)
- [Connect to an Offline Depot for a VCF Instance](/us/en/vmware-cis/vcf/vcf-9-0-and-later/9-0/lifecycle-management/lifecycle-management-of-vcf-core-components/configure-an-offline-depot-for-a-vcf-instance.html)

If you are connected to an offline depot, download the binaries to the offline depot. See [Download VCF Core Component Binaries to an Offline Depot](/us/en/vmware-cis/vcf/vcf-9-0-and-later/9-0/lifecycle-management/lifecycle-management-of-vcf-core-components/download-vcf-core-component-binaries-to-an-offline-depot.html).

1. In VCF Operations, click Fleet ManagementLifecycle.
2. Click VCF Instances and click the name of a VCF Instance.
3. Click Binary Management.
4. Select the VCF Version from the drop-down menu.
5. Click to select the type of binaries you want to download.

   - Upgrade Binaries
   - Patch Binaries
   - Install Binaries
6. Select one or more binaries to download and select an option.

   Option | Details || Download | Select this option to start downloading binaries now. |
   | Schedule Download | Select this option to schedule a date and time to start downloading binaries. |
