# Unassign an External IP

Learn how to unassign an external IP from your virtual machines.

1. In the vSphere Client, navigate to a VPC or a Subnet.
2. Click Configure > External IPs.
3. Select the IP listed under External IP.
4. Click Unassign.
5. Confirm the prompt by clicking Unassign.
