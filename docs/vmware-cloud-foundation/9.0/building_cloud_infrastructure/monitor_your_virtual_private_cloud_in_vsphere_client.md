# Monitor your Virtual Private Cloud in vSphere Client

Learn how to monitor VPCs and information related to the current state of each VPC.

1. To monitor VPCs in your vSphere Client, select Virtual Private Clouds, datacenter, a cluster, or a host from the inventory and navigate to Networks > Virtual Private Clouds folder.
2. Use the drop down menu to select a Quick Filter or an Advanced Filter.
   1. To use Quick Filter, enter value in the field.
   2. To use an advanced filter, select Advanced Filter from the drop-down menu and click **Add New** to enter the parameters for search.
