# View the Topology of Virtual Private Cloud in vCenter

View the structure and components of a Virtual Private Cloud by using its topology diagram. The topology diagram of a VPC provides a visual representation of the VPCs and Subnets connected to the switch.

1. In the vSphere Client, navigate to Virtual Private Clouds folder in your datacenter.
2. Select a VPC from the VPCs are listed under Virtual Private Clouds folder.
3. Select Configure > Topology.

The topology diagram of a VPC in is displayed.
