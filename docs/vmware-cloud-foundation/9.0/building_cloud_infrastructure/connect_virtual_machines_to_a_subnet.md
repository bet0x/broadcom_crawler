# Connect Virtual Machines to a Subnet

Learn how to connect virtual machines to a subnet. You can also connect VM to subnet by editing the VM.

1. From your browser, log in to the vSphere Client.
2. Navigate to Networking > VPC > VPC Subnets
3. Select a VPC Subnet from your inventory.
4. From the **Actions** drop-down menu, select Connect VMs.
5. You can filter using a Quick Filter or an Advanced Filter.
6. Select virtual machines to connect to the subnet.
7. Select a network adapter from the Network Adapter drop-down menu.
8. On the Ready to Complete page, click Finish.
