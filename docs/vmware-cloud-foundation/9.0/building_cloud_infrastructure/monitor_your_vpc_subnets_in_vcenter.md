# Monitor your VPC Subnets in vCenter

Learn how vSphere monitors VPC subnets and provides information about the current state of each VPC subnet.

1. To monitor VPCs in your vSphere Client, you can
2. Navigate to inventory and navigate to Networks > VPC Subnet

   or
3. Select a host, datacenter, cluster, VM, VPC, or a NSX Managed folder from the inventory and navigate to Networks > VPC Subnet
4. Use the drop down menu to select a Quick Filter or an Advanced Filter.
5. To use the advanced filter, select Advanced Filter from the drop-down menu and click **Add New** to enter the parameters for search.
