# Upgrade vSphere Distributed Switch Versions

[Optional] Upgrade the vSphere Distributed Switch in the vSphere Client to take advantage of features that are available only in the later versions.

ESX and vCenter upgrades are completed.

1. On the vSphere Client Home page, click Networking and navigate to the vSphere Distributed Switch.
2. Right-click the vSphere Distributed Switch and select UpgradeUpgrade Distributed Switch.
3. Select the vSphere Distributed Switch version that you want to upgrade the switch to and click Next.

The vSphere Distributed Switch is successfully upgraded.
