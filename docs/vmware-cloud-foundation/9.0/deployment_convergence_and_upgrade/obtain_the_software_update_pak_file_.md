# Obtain the Software Update PAK File

You upgrade your VCF Operations environment and cloud proxy using a PAK file.

## Download the PAK Files

You download the upgrade PAK file to upgrade your VCF Operations cluster. In case modifications are required, you can manually update the hosts file after completing the software update.

To download the PAK file for VCF Operations, go to [Broadcom Support Portal](https://support.broadcom.com/) and select the correct version from the drop-down list.

Download the VMware Cloud Foundation Operations PAK file from the Broadcom Support Portal, to update the VCF Operations environment and your cloud proxy together.
