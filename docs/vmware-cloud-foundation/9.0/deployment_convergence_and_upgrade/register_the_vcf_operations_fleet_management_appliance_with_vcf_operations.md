# Register the VCF Operations fleet management Appliance with VCF Operations

Use the VCF Operations admin UI to register the fleet management appliance.

Perform the following steps to register the fleet management appliance. The process takes about five minutes to complete.

1. Launch the VCF Operations admin UI, for example: https://vcfops-masternode-fqdn/admin/
2. Log in as admin user.
3. On System StatusFleet Management, click Connect.
4. To connect the fleet management node, provide:

   - Node address: Enter the FQDN of the VCF Operations fleet management appliance.
   - Admin Password: Enter the VCF Operations fleet management admin password.
5. Click Test Connection.
6. Review and accept the certificate. Click Next.
7. Enter the VCF Operations admin password.
8. To start the registration process, click Finish.
