# Add an NSX IP Address Block

You can configure
IP address blocks for use by other components.

1. With admin privileges, log in
   to NSX Manager.
2. Select NetworkingIP ManagementIP Address Pools.
3. Click the
   IP Address
   Blocks tab.
4. Click Add IP Address
   Block.
5. Enter a name and
   optionally a description.
6. Enter an IP block in
   CIDR format.
7. Click
   Save.
