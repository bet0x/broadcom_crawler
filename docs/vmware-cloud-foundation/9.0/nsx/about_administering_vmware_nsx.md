# About Administering VMware NSX

The NSX Administration Guide provides information about configuring and managing networking for VMware NSX (Formerly known as NSX-T Data Center), including how to create logical switches and ports and how to set up networking for tiered logical routers, configure NAT, firewalls, SpoofGuard, grouping and DHCP.

## Intended Audience

This information is intended for anyone who wants to configure NSX. The information is written for experienced Windows or Linux system administrators who are familiar with virtual machine technology, networking, and security operations.
