# NSX Edge Installation Methods

Install NSX Edge on an ESX host using NSX Manager UI (recommended method), vSphere Web Client (from UI or vSphere command-line OVF tool) or as physical severs.

## NSX Edge Installation Methods

| Installation Method | Instructions |
| --- | --- |
| NSX Manager (recommended method to install an NSX Edge VM appliance only) | - Ensure NSX Edge network requirements are met. - Create an NSX Edge transport node. See [Create an Transport Node](/us/en/vmware-cis/vcf/vcf-9-0-and-later/9-0/advanced-network-management/administration-guide/installing-nsx-edge/create-an-edge-transport-node.html). - Create an NSX Edge cluster. See [Create an Cluster](/us/en/vmware-cis/vcf/vcf-9-0-and-later/9-0/advanced-network-management/administration-guide/installing-nsx-edge/create-an-edge-cluster.html). |
| vSphere web client or vSphere command-line OVF tool | - Ensure NSX Edge network requirements are met. See [Installation Requirements](/us/en/vmware-cis/vcf/vcf-9-0-and-later/9-0/advanced-network-management/administration-guide/installing-nsx-edge/nsx-edge-installation-requirements.html). - Choose vSphere web client or vSphere command-line OVF tool to install NSX Edge.   - (Web Client) Install NSX Edge on ESX. See [Install an on Using the GUI](/us/en/vmware-cis/vcf/vcf-9-0-and-later/9-0/advanced-network-management/administration-guide/installing-nsx-edge/manually-deploying-nsx-edge-node-outside-of-nsx/install-an-nsx-edge-on-esxi-using-a-gui.html).   - (Command-line OVF tool) Install NSX Edge on ESX. See [Install on Using the Command-Line OVF Tool](/us/en/vmware-cis/vcf/vcf-9-0-and-later/9-0/advanced-network-management/administration-guide/scale-up-nsx-manager/installing-nsx-manager-cluster-on-vsphere/install-nsx-manager-on-esxi-using-the-command-line-ovf-tool.html).  - Join NSX Edge with the Management Plane. See [Join with the Management Plane](/us/en/vmware-cis/vcf/vcf-9-0-and-later/9-0/advanced-network-management/administration-guide/installing-nsx-edge/manually-deploying-nsx-edge-node-outside-of-nsx/join-nsx-edges-with-the-management-plane.html). - Configure an NSX Edge as a transport node. See [Edit Transport Node Configuration](/us/en/vmware-cis/vcf/vcf-9-0-and-later/9-0/advanced-network-management/administration-guide/installing-nsx-edge/manually-deploying-nsx-edge-node-outside-of-nsx/edit-nsx-edge-transport-node-configuration.html). - Create an NSX Edge cluster. See [Create an Cluster](/us/en/vmware-cis/vcf/vcf-9-0-and-later/9-0/advanced-network-management/administration-guide/installing-nsx-edge/create-an-edge-cluster.html). |
| Physical server (Automated or Interactive mode using ISO file) or NSX Edge VM appliance | Install NSX Edge using ISO file using PXE on physical servers. Note that PXE boot installation procedure is not supported on NSX Manager.   - Ensure NSX Edge network requirements are met.  - Join NSX Edge with the Management Plane. See [Join with the Management Plane](/us/en/vmware-cis/vcf/vcf-9-0-and-later/9-0/advanced-network-management/administration-guide/installing-nsx-edge/manually-deploying-nsx-edge-node-outside-of-nsx/join-nsx-edges-with-the-management-plane.html). - Configure an NSX Edge as a transport node. See [Edit Transport Node Configuration](/us/en/vmware-cis/vcf/vcf-9-0-and-later/9-0/advanced-network-management/administration-guide/installing-nsx-edge/manually-deploying-nsx-edge-node-outside-of-nsx/edit-nsx-edge-transport-node-configuration.html). - Create an NSX Edge cluster. See [Create an Cluster](/us/en/vmware-cis/vcf/vcf-9-0-and-later/9-0/advanced-network-management/administration-guide/installing-nsx-edge/create-an-edge-cluster.html). |
