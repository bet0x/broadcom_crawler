# Profiles

There are two types of profiles: context profiles and layer 7 access
profiles.

Profiles enable creating attributes key value
pairs such as layer 7 App ID, and domain names. After a profile has been defined, it can
be used in one or more distributed firewall rules, or gateway firewall rules.
