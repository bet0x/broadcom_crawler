# IPv6 Unsupported Features

NSX does not support a few
features if you select IPv6 as the addressing scheme while configuring any of these
objects or components:

- Host transport node
- NSX Edge transport node
- Transport node profile
- QuickStart card to configure
  NSX

The unsupported features are:

- EVPN
- Multicast Routing
- NSX Federation
