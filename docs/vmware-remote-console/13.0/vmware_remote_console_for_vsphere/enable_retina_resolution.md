# Enable Retina Resolution

Full resolution for Retina Display option makes VMRC less stretchable, but the virtual machine console appears sharper on a Retina display.

1. In a web browser, go to vmrc://settings and click Open Link. The Connection Proxy dialog appears. If VMRC was already opened:

   - On MacOS, select VMware Remote Console > Settings.
2. Click Display.
3. Check Use full resolution for Retina Display.
